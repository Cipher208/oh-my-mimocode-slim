# Contributing

1. Fork the repository
2. Create a branch: `git checkout -b feature/my-feature`
3. Make changes to hooks or skills
4. Test with MiMoCode
5. Commit and push
6. Open a Pull Request

## Adding Hooks

1. Create `.ts` file in `hooks/`
2. Export default object with hook handlers
3. Add to README.md
4. Test with MiMoCode

## Adding Skills

1. Create `.md` file in `skills/`
2. Add frontmatter (name, description)
3. Document the skill
4. Test with MiMoCode
